# PlantDoc 🐾 | AI Farming Assistant

**PlantDoc** is a simple, smart web application designed to help farmers instantly identify plant diseases and find effective remedies using AI.

## 🌿 Aim
To develop a user-friendly assistant that identifies plant diseases through a smartphone camera or photo upload and provides immediate diagnostic results and treatment advice.

## ✨ Features
* **Live AI Scan:** Use your webcam to scan leaves in real-time.
* **File Upload:** Upload existing photos for instant diagnosis.
* **Remedy Engine:** Get specific treatment advice for every detected disease.

## 🛠️ Built With
* **AI Model:** Trained using Google Teachable Machine.
* **Neural Engine:** TensorFlow.js for in-browser detection.
* **Frontend:** Responsive HTML5 and CSS3 (Plus Jakarta Sans).
* **Hosting:** GitHub Pages.

## 📂 Repository Structure
* `index.html`: The main dashboard and AI logic.
